## Run tests
```bash
pipenv run pytest
```

## Run linter
```bash
pipenv run lint
```

## Run tensorboard
```bash
pipenv run tensorboard
```

Will create a tensorboard instance at [http://localhost:6006/](http://localhost:6006/)
